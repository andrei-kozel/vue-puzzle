# vue-puzzle
15 puzzle game

![](https://github.com/andrei-kozel/vue-puzzle/raw/master/public/puzzle.PNG | width=200)

[DEMO](https://vue-puzzle.netlify.com/)

## Project setup
```
yarn install
```
or
```
npm install 
```

### Compiles and hot-reloads for development
```
yarn serve 
```
or
```
npm run serve 
```
### Compiles and minifies for production
```
yarn build
```
or
```
npm run build 
```
